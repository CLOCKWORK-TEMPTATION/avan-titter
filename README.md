# محرر السيناريو العربي المتقدم

محرر سيناريو متطور مصمم خصيصاً للكتابة العربية مع دعم كامل للاتجاه من اليمين إلى اليسار (RTL).

## المميزات

- ✅ محرر سيناريو متقدم مع تنسيق تلقائي
- ✅ دعم كامل للغة العربية والاتجاه RTL
- ✅ أدوات ذكية للبحث والاستبدال
- ✅ مساعد الكتابة بالذكاء الاصطناعي
- ✅ إحصائيات مباشرة للنص
- ✅ تصدير متعدد الصيغ
- ✅ الوضع الليلي/النهاري
- ✅ حفظ تلقائي
- ✅ وكلاء ذكية متقدمة

## التثبيت

```bash
npm install
```

## التشغيل

### وضع التطوير
```bash
npm run dev
```

### البناء للإنتاج
```bash
npm run build
npm start
```

## الاستخدام

1. افتح المحرر في المتصفح
2. ابدأ الكتابة مباشرة
3. استخدم Tab للتنقل بين أنواع التنسيق
4. استخدم Ctrl+S للحفظ
5. استخدم Ctrl+F للبحث

## اختصارات لوحة المفاتيح

- `Tab` - التنقل للتنسيق التالي
- `Shift+Tab` - التنقل للتنسيق السابق
- `Ctrl+1-6` - تطبيق تنسيق محدد
- `Ctrl+F` - البحث
- `Ctrl+H` - البحث والاستبدال
- `Ctrl+S` - الحفظ

## أنواع التنسيق

1. **عنوان المشهد العلوي** - معلومات المؤلف والتاريخ
2. **عنوان المشهد** - رقم المشهد ووصفه
3. **وصف الأفعال** - وصف ما يحدث في المشهد
4. **الشخصية** - اسم الشخصية المتحدثة
5. **الحوار** - كلام الشخصية
6. **الانتقال** - انتقالات بين المشاهد
